## 亮点汇总

### 数据库
- FTS5 原子同步（Create/Update/Delete/Restore）已全面覆盖
- 迁移系统 8 版本全部事务包裹、幂等保护
- 参数化查询全覆盖，零 SQL 注入

### AI/检索
- 三路检索 + 加权 RRF 融合 + MMR 重排在同类项目中一流
- 查询预处理器 CJK 语言感知阈值精细
- 三级 fallback 解析策略健壮

### 安全
- TeamID/OwnerID 强制注入防伪造
- 防止系统身份冒充（__system__ 降级）
- 错误响应无内部细节泄露

### 架构
- bootstrap 包统一初始化，两入口复用
- Feature gating（nil 检查）一致且完善
- Scheduler atomic.Bool 零锁重叠防护
- Sentinel errors 体系清晰
