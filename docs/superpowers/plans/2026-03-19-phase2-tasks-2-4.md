# Phase 2 Tasks 2-4 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement Extractor (auto entity extraction), Graph-Enhanced Retrieval (three-way weighted RRF), and Token-Aware Trimming for IClude's Phase 2 intelligent memory system.

**Architecture:** Three features layered on top of Phase 1's existing Manager/Retriever/GraphStore and Task 1's `internal/llm/` package. Extractor adds LLM-based entity extraction during memory creation. Graph Retrieval adds a third RRF channel using entity associations. Token Trimming adds budget-based result truncation at the retrieval pipeline's tail.

**Tech Stack:** Go 1.25+, SQLite (FTS5), Qdrant, Gin, Zap, OpenAI-compatible LLM API

// RRFInput 加权RRF输入 / Weighted RRF input
type RRFInput struct {
	Results []*model.SearchResult
	Weight  float64
}

// MergeWeightedRRF 加权RRF融合 / Weighted RRF fusion
// score(id) = Σ weight × 1/(k + rank + 1)
func MergeWeightedRRF(inputs []RRFInput, k int, limit int) []*model.SearchResult {
	scores := make(map[string]float64)
	seen := make(map[string]*model.SearchResult)

	for _, input := range inputs {
		if input.Weight == 0 {
			continue
		}
		for rank, r := range input.Results {
			id := r.Memory.ID
			scores[id] += input.Weight * (1.0 / float64(k+rank+1))
			if _, ok := seen[id]; !ok {
				seen[id] = r
			}
		}
	}

	// 创建新对象，不修改输入 / Create new objects, don't mutate input
	results := make([]*model.SearchResult, 0, len(seen))
	for id, r := range seen {
		results = append(results, &model.SearchResult{
			Memory: r.Memory,
			Score:  scores[id],
			Source: "hybrid",
		})
	}

	sort.Slice(results, func(i, j int) bool {
		return results[i].Score > results[j].Score
	})

	if len(results) > limit {
		results = results[:limit]
	}
	return results
}
```

// estimateTokens 估算文本token数 / Estimate token count for text
func estimateTokens(text string) int {
	return len([]rune(text))
}

// trimByTokenBudget 按token预算裁剪检索结果 / Trim search results by token budget
func trimByTokenBudget(results []*model.SearchResult, maxTokens int) ([]*model.SearchResult, int, bool) {
	if maxTokens <= 0 || len(results) == 0 {
		total := 0
		for _, r := range results {
			total += estimateTokens(r.Memory.Content)
		}
		return results, total, false
	}

	var trimmed []*model.SearchResult
	totalTokens := 0
	for i, r := range results {
		tokens := estimateTokens(r.Memory.Content)
		if totalTokens+tokens > maxTokens && i > 0 {
			return trimmed, totalTokens, true
		}
		trimmed = append(trimmed, r)
		totalTokens += tokens
	}
	return trimmed, totalTokens, false
}
```
